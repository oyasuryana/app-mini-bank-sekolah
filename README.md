# Instalasi
1. Buat database dengan nama apl_tabungan 
2. Import file apl_tabungan.sql
3. Edit file konfigurasi.php

	$user	='root';
	$pass	='';
	$host	='localhost';
	$db		='apl_tabungan';

4. Simpan 
5. Jalankan
